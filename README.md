DZT
===

[![Build Status](https://travis-ci.org/dblock/dzt.png)](https://travis-ci.org/dblock/dzt)

Slice deep-zoom tiled images.

## Usage

```
gem install dzt
```

#### Get Help

```
dzt help
```

#### Tile an Image

```
dzt slice image.jpg --output tiles
```

Creates a *tiles* folder with deep-zoom tiles.
